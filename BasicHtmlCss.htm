HTML Basics

  • What is HTML?

    HTML is the standard markup language for creating Web pages.

    • HTML stands for Hyper Text Markup Language
    • HTML describes the structure of Web pages using markup
    • HTML elements are the building blocks of HTML pages
    • HTML elements are represented by tags
    • HTML tags label pieces of content such as "heading", "paragraph", "table", and so on
    • Browsers do not display the HTML tags, but use them to render the content of the page

  • Why do we use tags?

    Tags are how the computer knows how to we want it to format and structure the information displayed on the webpage.

  • Open and Close

    • (Almost) every HTML element has an opening (<element>) and a closing (</element>) tag that surrounds the information affected by that element
    • Elements can be nested inside each other (<a><b>Something</b></a>)
    • When it finds an opening tag, the computer applies that element to everything that follows until it reaches a matching closing tag
    • Many errors in how a webpage displays can be traced back to a missing or misplaced opening or (most often) closing tag

  • Structure Elements

    • Headings

      • Headings come in 6 sizes: <h1>, <h2>, <h3>, <h4>, <h5>, and <h6>
      • These are useful for titles and beginnings of sections
      • Larger headings can be used for larger sections
      • Smaller headings can be used for smaller sections or subsections
      • Samples:

        • Heading 1!

        • Heading 2!

        • Heading 3!

        • Heading 4!

        • Heading 5!
        • Heading 6!

    • Paragraphs

      • Paragraphs are basic blobs of text
      • These can be used to chunk related information into groups
      • The computer automatically adds white space (margin) around a paragraph element
      • The computer ignores newlines (return/enter) and extra spaces inside a paragraph element
